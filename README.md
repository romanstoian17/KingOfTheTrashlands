# King of the Trashlands

First playable Roblox foundation for a PvP arena game with 16 safe bases, a central combat arena, an underground mob area, prototype combat classes and abilities, and a timed boss loop.

## Run

This is a Rojo-style project. Sync `default.project.json` into Roblox Studio, then press Play. The server generates the test map at runtime.

## Current gameplay

- Players spawn in one of 16 outer bases.
- Bases are safe zones. PvP and NPC damage are ignored when either side is protected.
- Players choose between Fire Caster, Ice Mage, and Lightning Mage, each with starter ability tools.
- The central arena allows PvP.
- The subway under the arena contains respawning mobs.
- A boss spawns in the center every random 5-10 minutes and rewards contributors with placeholder TrashCoins.
